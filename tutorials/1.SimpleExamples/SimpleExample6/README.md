On this example we show you how to define the a Popup Menu for a Control.
		
```python
self._fullname.addPopupSubMenuOption('Path', 
	{
		'Delete':           self.__dummyEvent, 
		'Edit':             self.__dummyEvent,
		'Interpolate':      self.__dummyEvent
	})
```
